# Тестовое задание для стажера в группу вычислительной химии Антюшова А.А.

# Задание 1. Параметризация малой молекулы полем OpenFF

Написать набор функций в виде Python-скрипта или Jupyter-ноутбука для параметризации большого количества малых молекул полем OpenFF для использования в Gromacs.

- Чтение нескольких лигандов из SDF
- Сохранение исходных координат молекул
- Генерация файлов:
  - структуры лиганда (`ligand.pdb`)
  - топологии Gromacs (`topol.top`)
  - координат Gromacs (`conf.gro`)
- Автоматическая обработка большого количества молекул
- Создание отдельной папки для каждого лиганда

## Используемые библиотеки

- Python 3.11
- RDKit
- OpenFF Toolkit
- ParmEd

## Установка окружения

```bash
conda create -n openff python=3.11 -y
conda activate openff

mamba install -c conda-forge \
    rdkit \
    openff-toolkit \
    parmed \
    -y
```

## Использование

Поместить файл:

```text
example.sdf
```

в папку со скриптом и запустить:

```bash
python openff_parametrize.py
```

## Структура выходных данных

```text
output/
  ── CHEMBL45305/
       ── ligand.pdb
       ── conf.gro
       ── topol.top
```

## Особенности реализации

- Координаты молекул сохраняются без изменений
- Для расчёта частичных зарядов используются заряды Gasteiger (RDKit)
- Ошибки отдельных молекул не прерывают выполнение всего пайплайна
- Поддерживается пакетная обработка большого числа лигандов
